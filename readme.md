## Compatibility

- Linux
- Windows
- Mac

## Installing

Install globally:

    npm install -g sextadoscria

## Running

Run command:

    sextadoscria

Or import in NodeJS script:

    import daledale from 'sextadoscria';

    sextadoscria();

## Running with [npx](https://www.npmjs.com/package/npx)

Run command:

    npx sextadoscria
